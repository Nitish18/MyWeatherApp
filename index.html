MyWeatherApp

Android Application for finding weather conditions for 5 cities currently :- 1.) Jaipur 2.) Banglore 3.) Bikaner 4.) Delhi 5.) Mumbai

The app uses the weather api :- "http://developers.forecast.io" Technology used:- OkHttp,ButterKnife,JSON,Spinners.
